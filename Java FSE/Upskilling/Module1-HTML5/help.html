How to Register for an Event

  1. Open the Event Registration section.
  2. Enter your name and email.
  3. Select an event date.
  4. Choose an event type.
  5. Click Register.

Frequently Asked Questions

1. Is registration free?

Yes, registration is free for all residents.

2. Can I cancel my registration?

Yes, contact the support team.

3. How can I contact support?

Email: support@communityportal.com
